Woocommerce Exclude Emails Coupon is a simple Woocomerce extension that allows to exclude emails from having access to specific coupons.

== Description ==

Exclude Emails Coupon provides a solution for preventing 
specific emails from using a WooCommerce coupon.

== Installation ==

1. Copy the 'woocommerce-exlude-emails-coupon' folder into your plugins folder
2. Activate the plugin via the Plugins admin page
